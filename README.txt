Fish Predictor PRO v4

Cara menjalankan:
1. Extract file ZIP
2. Masuk ke folder hasil extract
3. Jalankan server:
   python -m http.server 7700
4. Buka di browser:
   http://localhost:7700

Fitur:
- GPS otomatis
- Cari lokasi
- Klik peta
- Forecast aktivitas ikan 48 jam
- Grafik aktivitas 48 jam
- Moon phase
- Solunar window berbasis sunrise & sunset
- Pilihan target ikan
- Pilihan tipe perairan
- Simpan spot favorit

Catatan:
- Ini scoring engine heuristik yang lebih kaya, belum machine learning penuh.
- Untuk akurasi lebih tinggi lagi, versi berikutnya bisa ditambah log strike, pasang surut laut spesifik, suhu air, dan histori spot.
